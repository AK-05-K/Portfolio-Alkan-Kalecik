Language learning app
Description
Built a full-stack Duolingo-inspired language learning application in a university team (group of 4) project using agile development practices. Camera-Based Game Control System
Description
In this university project, which was part of the Human-Computer Interaction (HCI) module, we were provided with an existing Super Mario codebase. The task was to extend the game by integrating camera-based movement detection, enabling real-time control of the game through physical body movements. The project focused on processing live camera input, detecting user movement, and mapping these inputs to in-game actions in a responsive and interactive way.
